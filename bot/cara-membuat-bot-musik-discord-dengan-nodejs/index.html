Cara Membuat Bot Musik Discord Dengan Node.js

Oct 12, 2021
Waktu Membaca: 2 menit

Halo bro selamat datang kembali di blog Made WGM kali ini saya akan membahas bagaimana caranya membuat bot discord musik dengan menggunakan node.js,

Pertama-tama yang perlu agan siapkan adalah node JS, git , YouTube api dan juga bot token nya

Pertama-tama yang harus kamu lakukan untuk membuat melodis adalah clone dulu repo nya

git clone https://github.com/eritislami/evobot.git && cd  evobot

Setelah itu silahkan buat bot discord baru Pertama tama buatlah aplikasi Bot Discord dengan memasuki link berikut ini https://discordapp.com/developers/applications/me . Kemudian klik New Application ‚Äì masukan nama bot kamu ‚Äì klik Create .

Pilih nama yang tepat untuk aplikasi

Setelah itu, kita perlu memilih tab bot dan klik tambahkan bot dan copy token dan simpan, kita akan membutuhkannya nanti.

Sekarang setelah aplikasi dan bot dibuat, kamu harus mengaitkan bot ke server kami

Untuk cara mendapatkan YouTube api silakan baca di sini https://developers.google.com/youtube/v3/getting-started Jika bahan bahan sudah di sediakan Sekarang tinggal membuat config.json nya aja

Silakan Jalankan perintah

mv config.json.example config.json

Jika sudah silakan buka teks Editor dan edit config.json nya

{
  "TOKEN": "",
  "YOUTUBE_API_KEY": "",
  "SOUNDCLOUD_CLIENT_ID": "",
  "MAX_PLAYLIST_SIZE": 10,
  "PREFIX": "/",
  "PRUNING": false,
  "LOCALE": "en",
  "DEFAULT_VOLUME": 100,
  "STAY_TIME": 30
}

Jika sudah di edit langsung jalankan perintah

npm install

Dan

node index.js
